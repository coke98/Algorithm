### 💡문제 분석

- 두 사람이 선물을 주고받은 기록이 있다면, 이번 달까지 두 사람 사이에 더 많은 선물을 준 사람이 다음 달에 선물을 하나 받습니다.
- 두 사람이 선물을 주고받은 기록이 하나도 없거나 주고받은 수가 같다면, 선물 지수가 더 큰 사람이 선물 지수가 더 작은 사람에게 선물을 하나 받습니다.
- 선물 지수는 이번 달까지 자신이 친구들에게 준 선물의 수에서 받은 선물의 수를 뺀 값입니다.
- 만약 두 사람의 선물 지수도 같다면 다음 달에 선물을 주고받지 않습니다.
- 당신은 선물을 가장 많이 받을 친구가 받을 선물의 수를 알고 싶습니다.

### 입력

- 이름을 담은 1차원 문자열 배열 friends, (2 ≤ friends의 길이 = 친구들의 수 ≤ 50)
- 이번 달까지 친구들이 주고받은 선물 기록을 담은 1차원 문자열 배열 gifts, (1 ≤ gifts의 길이 ≤ 10,000)

### 출력

- 다음달에 가장 많은 선물을 받는 친구가 받을 선물의 수를 return

---

### 💡입출력 예시

| friends | gifts | result |
| --- | --- | --- |
| ["muzi", "ryan", "frodo", "neo"] | ["muzi frodo", "muzi frodo", "ryan muzi", "ryan muzi", "ryan muzi", "frodo muzi", "frodo ryan", "neo muzi"] | 2 |
| ["joy", "brad", "alessandro", "conan", "david"] | ["alessandro brad", "alessandro joy", "alessandro conan", "david alessandro", "alessandro david"] | 4 |
| ["a", "b", "c"] | ["a b", "b a", "c a", "a c", "a c", "c a"] | 0 |

---

### 💡알고리즘 설계

- dic 활용

### 💡코드

```python
def solution(friends, gifts):
    answer = 0
    dic = {}
    for gift in gifts:
        A, B = gift.split()
        dic[gift] = dic[gift] + 1 if gift in dic else 1
        dic[A] = dic[A] + 1 if A in dic else 1
        dic[B] = dic[B] - 1 if B in dic else -1 
        
    print(dic)
    
    for I in friends:
        given = 0
        for friend in friends:
            if I == friend:
                continue
            print(f'{I} -> {friend}')
            
            dic[f'{I} {friend}'] = dic[f'{I} {friend}'] if f'{I} {friend}' in dic else 0
            dic[f'{friend} {I}'] = dic[f'{friend} {I}'] if f'{friend} {I}' in dic else 0
            dic[I] = dic[I] if I in dic else 0
            dic[friend] = dic[friend] if friend in dic else 0
            
            if dic[f'{I} {friend}'] > dic[f'{friend} {I}']:
                given += 1
                print('!')
            if dic[f'{I} {friend}'] == dic[f'{friend} {I}']:
                if dic[I] > dic[friend]:
                    given +=1
                    print('!')
        print(given)
        answer = max(answer, given)
    return answer
```
