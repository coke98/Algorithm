도넛 모양 그래프, 막대 모양 그래프, 8자 모양 그래프가 여러 개 있습니다. 이 그래프들과 무관한 정점을 하나 생성한 뒤, 각 도넛 모양 그래프, 막대 모양 그래프, 8자 모양 그래프의 임의의 정점 하나로 향하는 간선들을 연결했습니다.

그 후 각 정점에 서로 다른 번호를 매겼습니다.

이때 당신은 그래프의 간선 정보가 주어지면 생성한 정점의 번호와 정점을 생성하기 전 도넛 모양 그래프의 수, 막대 모양 그래프의 수, 8자 모양 그래프의 수를 구해야 합니다.

그래프의 간선 정보를 담은 2차원 정수 배열 `edges`가 매개변수로 주어집니다. 이때, 생성한 정점의 번호, 도넛 모양 그래프의 수, 막대 모양 그래프의 수, 8자 모양 그래프의 수를 순서대로 1차원 정수 배열에 담아 return 하도록 solution 함수를 완성해 주세요.

---

### 💡입출력 예시

| edges | result |
| --- | --- |
| [[2, 3], [4, 3], [1, 1], [2, 1]] | [2, 1, 1, 0] |
| [[4, 11], [1, 12], [8, 3], [12, 7], [4, 2], [7, 11], [4, 8], [9, 6], [10, 11], [6, 10], [3, 5], [11, 1], [5, 3], [11, 9], [3, 8]] | [4, 0, 1, 2] |

### 💡알고리즘 설계

- 각 정점에 간선 구조를 분석해서 그래프 모양, 개수를 유추
- 도넛의 경우 직선, 8자를 전체에서 제외하여 계산

---

### 💡코드

```python
def solution(edges):
    answer = []
    ie = {}
    oe = {}
    nodes = set()
    new_node = 0
    graph_cnt = 0
    straight_cnt = 0
    eight_cnt = 0
    for edge in edges:
        o = edge[0]
        i = edge[1]
        ie[i] = ie[i] + 1 if i in ie else 1
        oe[o] = oe[o] + 1 if o in oe else 1
        nodes.add(i)
        nodes.add(o)
    for x in nodes:
        ie[x] = ie[x] if x in ie else 0
        oe[x] = oe[x] if x in oe else 0
        
        # 생성된점
        if oe[x] >= 2 and ie[x] == 0:
            new_node = x
            graph_cnt = oe[x]
        # 막대 개수
        if oe[x] == 0:
            straight_cnt += 1
        
        # 8자 개수
        if oe[x] == 2 and ie[x] >= 2:
            eight_cnt += 1
    
    # 도넛 개수
    donut_cnt = graph_cnt - straight_cnt - eight_cnt
    
    answer = [new_node, donut_cnt, straight_cnt, eight_cnt]    
    return answer
```
