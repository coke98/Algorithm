### 💡문제 분석

A와 B가 `n`개의 주사위를 가지고 승부를 합니다. 주사위의 6개 면에 각각 하나의 수가 쓰여 있으며, 주사위를 던졌을 때 각 면이 나올 확률은 동일합니다. 각 주사위는 1 ~ `n`의 번호를 가지고 있으며, 주사위에 쓰인 수의 구성은 모두 다릅니다.

A가 먼저 `n / 2`개의 주사위를 가져가면 B가 남은 `n / 2`개의 주사위를 가져갑니다. 각각 가져간 주사위를 모두 굴린 뒤, 나온 수들을 모두 합해 점수를 계산합니다. 점수가 더 큰 쪽이 승리하며, **점수가 같다면 무승부**입니다.

A는 자신이 **승리할 확률**이 가장 높아지도록 주사위를 가져가려 합니다.

---

### 💡입출력 예시

| dice | result |
| --- | --- |
| [[1, 2, 3, 4, 5, 6], [3, 3, 3, 3, 4, 4], [1, 3, 3, 4, 4, 4], [1, 1, 4, 4, 5, 5]] | [1, 4] |
| [[1, 2, 3, 4, 5, 6], [2, 2, 4, 4, 6, 6]] | [2] |
| [[40, 41, 42, 43, 44, 45], [43, 43, 42, 42, 41, 41], [1, 1, 80, 80, 80, 80], [70, 70, 1, 1, 70, 70]] | [1, 3] |

### 💡알고리즘 설계

- 콤비네이션 + dfs + 이분탐색

### 💡코드

- 개선한 코드

```python
from itertools import combinations
from collections import deque
def solution(dice):
    pidx = 0
    answer = []
    max_win = 0
    n = len(dice)
    dc = [i for i in range(n)]
    t = [0,1,2,3,4,5]
    for dice_combo in combinations(dc,n//2):
        B_dice_combo = []
        for i in range(n):
            if i not in dice_combo:
                B_dice_combo.append(i)
        # print(f'A: {dice_combo} B: {B_dice_combo}')
                
        stack = []
        stack.append([])
        A = []
        while stack:
            l = stack.pop()
            idx = len(l)
            if idx == n//2:
                A.append(sum(l))
                continue
            for i in range(6):
                c_dice = dice_combo[idx]
                nl = l + [dice[c_dice][i]]
                stack.append(nl)
        A.sort()
                
        stack.append([])
        B = []
        while stack:
            l = stack.pop()
            idx = len(l)
            if idx == n//2:
                B.append(sum(l))
                continue
            for i in range(6):
                c_dice = B_dice_combo[idx]
                nl = l + [dice[c_dice][i]]
                stack.append(nl)
        B.sort()
        
        # if pidx == 0:
        #     print(A)
        #     print(B)
        #     pidx += 1
        win = 0
        for a in A:
            start = 0
            end = len(B) - 1
            result_index = 0

            while start <= end:
                mid = (start + end) // 2
        
                if B[mid] < a:
                    result_index = mid
                    start = mid + 1
                else:
                    end = mid - 1
            win += result_index + 1
        # print(win)
        # print(dice_combo)
        max_win = max(max_win, win)
        if max_win == win:
            answer = []
            for a_d in dice_combo:
                answer.append(a_d+1)
            # print(f'a: {a}')
            # print(result_index)
    return answer
```
